  About the Models Used

  ResNet50 (Residual Networks)

ResNet50 is considered a Very-Deep Convolutional Neural Network, meaning it consists of several layers that allow for deeper processing of images. This does not always equate to more accurate detection due to the risk of overfitting increasing with more layers. ResNet50 employs a "skip" connect to help reduce gradient loss, upholding higher accuracies. The end result is a lighter model with great ability to detect detail without teh risk of extreme overfitting.


  ResNet152 (Residual Networks)

ResNet152 is a larger version of the predecessor, ResNet50, using 152 layers for better image clarity. Using this model produced the highest accuracies of any of the 4 models but showed little change in prediction accuracies of images. This model does run the risk of overfitting more-so than ResNet50 due to the larger layer count but if tuned correctly and fed with sufficent data, can be nearly perfect in its output.


  5-Layer CNN

The simple 5-layer CNN used is a simple Neural Network that makes use of 2 Convolutional layers, 2 MaxPooling Layers, and a flattening layer. The purpose behind this model is to give an idea as to what is happening underneath the Very-Deep layer CNNs while still producing results. Since this model lacks depth, many of the results are near 100% confident, even if the answer is completely wrong. This model suffers from overfitting due to the lack of data diversity and the ability for it to detect smaller changes in pixel colors.


  VGG16

VGG16 is a powerful and deep-layer Neural Network capable of pulling out the small pixel details in an image. Similar to ResNet152 and ResNet50, VGG16 is also a winning model, taking both first and second place in the ILSVRC Challange (ImageNet Large Scale Visual Recognition Challenge). VGG16 gradually gets more intense in each layer, eventually falling into convolutional layers with channel sizes of 512.
